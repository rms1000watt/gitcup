# gitcup

Wrapper of `git commit -m "Updated"`

## Installation

```
npm install -g nimstall
nimstall -g gitcup
```

## Usage

### Command Line

```
gitcup
```

## Common Workflow

First install..

```
nimstall -g gitadot
```

Then do..

```
mkdir my-project
cd my-project
git init
touch Readme.md
gitadot
gitcup
git push
```
